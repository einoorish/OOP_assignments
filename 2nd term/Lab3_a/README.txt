Реалізувати алгоритми сортування для заданої структури даних (список
1). Реалізувати наступні алгоритми:
1) простий алгоритм (SelectionSort);
2) більш ефективний алгоритм quicksort (Lomuto partition scheme, rightmost pivot);
3) більш ефективний алгоритм merge sort (bottom-up);
4) комбінований алгоритм, який використовує більш ефективний алгоритм (один з quicksort чи merge sort) для великих масивів і простий алгоритм для маленький масивів;
5) сортування з використанням бібліотечних функцій.
